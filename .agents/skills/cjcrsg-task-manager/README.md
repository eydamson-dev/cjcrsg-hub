# CJCRSG Task Manager

Smart task management for CJCRSG-Hub with standardized naming and collaborative planning.

## Quick Usage

### Check Status

```
User: What's next?
AI: Latest: ✅ Phase 13 Complete
    Next: ⏳ Phase 14: Attendance Reporting
```

### Start Working

```
User: Start task
AI: Next up: Task 14.1: Create database schema
    Start this task? [Confirm]
```

### Complete Task

```
User: Task complete
AI: Moved to completed. Next: Task 14.2: Backend queries
```

### Plan New Phase

```
User: Plan Phase 14: Attendance Reporting
AI: What should this include? (I'll suggest breakdown)
    ...
    Create plan? [Confirm]
```

## Naming Standard

```
Phase X: Name           (main sections)
Task X.Y: Name          (individual tasks)
```

## Features

- ✅ Standardized naming enforcement
- ✅ Smart file detection (TASKS.md vs TDD_TASKS.md)
- ✅ UI-first ordering with dependency tracking
- ✅ Collaborative planning mode
- ✅ Auto-suggest file paths
- ✅ Dual placement (Next Up + Phase sections)
